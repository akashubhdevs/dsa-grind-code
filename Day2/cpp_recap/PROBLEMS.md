### Basics & Conditional Statements

1. The Valid Triangle: You are given three integer angles A, B, and C. First, check if they can form a valid triangle. If they can, output whether the triangle is Equilateral, Isosceles, or Scalene.

Hint: Don't forget to check if the angles add up correctly and that no angle is 0!

2. The Quadrant Checker: Given two integers X and Y representing a coordinate point on a 2D plane, output which quadrant it belongs to (Q1, Q2, Q3, or Q4). If it doesn't belong to a quadrant, specify if it lies on the X-axis, Y-axis, or exactly at the Origin.

### Loops
3. The Collatz Sequence (3N + 1): Take a positive integer N from the user. If N is even, divide it by 2. If N is odd, multiply it by 3 and add 1. Repeat this process and print the sequence of numbers until N becomes 1. At the end, output the total number of steps it took.
